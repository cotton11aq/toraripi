# トラリピ資金管理表

資金がいくらあればロスカットされないのか、一目ですぐにわかる計算アプリです。

FXにおいてロスカットレートの設定は非常に大切です。  
このアプリでは、ロスカットレートや購入金額を記入するだけで、ロスカットされないために必要な資金を計算することができます。  
また、計算結果をデータとして保存することができ、今いくら資金が拘束されているのか簡単にわかります。
